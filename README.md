# AI Robot Safety Edge AI Project

This repository documents my hands-on practice from the AI Robot Safety Control Solution program.

The project focuses on building an edge AI pipeline using Raspberry Pi 5, OpenCV, and YOLO11n.  
The final goal is to understand how camera-based object detection can be used as a foundation for industrial safety monitoring systems.

## Project Goals

- Set up Raspberry Pi 5 as an edge AI device
- Connect to the device using SSH and VNC
- Capture webcam frames with OpenCV
- Run real-time object detection using YOLO11n
- Measure FPS and latency on Raspberry Pi 5
- Filter detection results by target class
- Prepare the baseline for future safety monitoring applications
