# Day 1 - Raspberry Pi Environment Setup

## Goal

The goal of Day 1 was to set up Raspberry Pi as an edge AI device for the AI robot safety control project.

## What I Did

- Installed and configured Raspberry Pi OS
- Prepared the SD card for Raspberry Pi booting
- Connected Raspberry Pi to the network
- Checked the IP address of Raspberry Pi
- Set up remote access using SSH and VNC
- Practiced basic Linux commands
- Prepared the basic Python development environment

## Meaning

This step was necessary because the final safety monitoring system should run on a small edge device, not only on a laptop or cloud server.

By setting up Raspberry Pi, I prepared the hardware and software environment for a computer vision-based safety monitoring system.

## Project Interpretation

In this project, Raspberry Pi works as the main edge device that receives camera input, runs AI-based detection code, and processes safety-related information locally.

Therefore, Day 1 was the foundation stage for the full safety monitoring pipeline.

## What I Learned

- How to install and boot Raspberry Pi OS
- How to connect Raspberry Pi to a network
- How to access Raspberry Pi remotely using SSH or VNC
- How to use basic Linux commands
- Why Raspberry Pi can be used as an edge AI device

## Connection to the Final Project

The final project aims to detect unsafe situations, such as helmet or no-helmet cases, using camera input and computer vision.

Day 1 prepared the device environment required for this system.





# 1일차 - Raspberry Pi 개발 환경 구축

## 목표

1일차의 목표는 라즈베리파이를 AI 로봇 안전제어 프로젝트에 사용할 수 있는 Edge AI 장치로 준비하는 것이었다.

## 수행한 작업

- Raspberry Pi OS 설치 및 설정
- SD카드 준비 및 부팅 환경 구성
- 라즈베리파이 네트워크 연결
- IP 주소 확인
- SSH 및 VNC를 통한 원격 접속 환경 구축
- 기본 Linux 명령어 실습
- Python 개발 환경 준비

## 의미

이번 단계는 단순히 라즈베리파이를 설치하는 과정이 아니라, 이후 카메라 입력과 AI 모델을 실행하기 위한 기반 환경을 구축하는 과정이었다.

라즈베리파이는 최종 시스템에서 카메라 영상을 입력받고, AI 모델을 실행하며, 안전모 착용 여부와 같은 안전 관련 정보를 현장에서 처리하는 Edge Device 역할을 한다.

따라서 1일차는 전체 안전 모니터링 시스템의 하드웨어 및 소프트웨어 기반을 마련한 단계라고 할 수 있다.

## 배운 점

- 라즈베리파이를 사용하기 위해서는 운영체제 설치, 네트워크 연결, 원격 접속 설정이 필요하다는 것을 알게 되었다.
- SSH와 VNC를 통해 라즈베리파이를 직접 모니터에 연결하지 않고도 제어할 수 있었다.
- Linux 명령어를 사용해 폴더와 파일을 관리하는 기본적인 방법을 익혔다.
- 라즈베리파이가 현장 기반 AI 시스템에서 Edge Device로 활용될 수 있다는 점을 이해했다.

## 최종 프로젝트와의 연결

최종 프로젝트는 카메라 입력을 기반으로 안전모 착용 여부를 판단하는 AI 안전 모니터링 시스템을 만드는 것이다.

1일차에는 이 시스템이 실제로 실행될 라즈베리파이 환경을 구축하였다.
